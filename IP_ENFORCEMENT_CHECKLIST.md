# IP Enforcement Checklist

Use this when you find a copied app, copied source, or unauthorized redistribution.

## 1) Confirm and document

- Capture screenshots of the infringing app/page.
- Record URL(s), developer name/account, package name, and listing ID.
- Note the first date/time seen.
- Save comparison evidence (copied text, identical UI, same assets, same behavior).
- Save your ownership proof: repository link, commit history, NOTICE, LICENSE, release notes.

## 2) Preserve evidence

- Export the infringing listing page as PDF.
- Download any infringing APK if legally permitted in your jurisdiction.
- Keep file hashes and timestamps.
- Store evidence in a dated folder (for example `evidence/YYYYMMDD_case_name/`).

## 3) Send platform takedown first

- File notice through the platform process (Google Play, hosting provider, code host).
- Keep tone factual and specific.
- Include exact copied elements and links.
- Attach ownership proof and your contact details.

## 4) Track response window

- Set reminders for 48 hours, 7 days, and 14 days.
- Log every message and ticket ID.
- If unresolved, escalate within platform policy.

## 5) Escalate if needed

- Send formal cease-and-desist letter.
- Consider legal counsel for repeat or commercial infringement.
- Consider trademark filing for app name/logo if brand confusion is occurring.

## 6) Prepare your app/repo for easier enforcement

- Keep `NOTICE` and `LICENSE` current.
- Keep in-app About/Legal text current.
- Keep release artifacts with hashes.
- Keep dated backups of source and APKs.

## Quick evidence pack (minimum)

- Your app listing or release artifact + hash
- Infringing listing URL and screenshots
- Side-by-side comparison images
- NOTICE + LICENSE copies
- Statement of good-faith belief and accuracy
